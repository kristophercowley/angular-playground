Welcome to the Angular Playground

The angular playground is a simple tool to help get you introduced to angulars design patterns and quirks. Along the way you will likely find things you enjoy doing the JQuery way but the overall framework and organization that Angular brings to the table is well worth the learning curve.

With the power and speed that angular lends to us as developers when can quickly start writting full scale applications all while keeping our code clean and organized.

Angular offers many other benefits than just the ones discussed in this playground for more information read up at Angularjs.org

If you have not done it already please go through the CodeSchool Shaping up with Angular Course it is awesome.

Exercises

The angular playground is a teaching tool so feel free to break things. You will notice that each section of the playground has notifications as to the status of the playground. If something is broken you will likely know with the help of the notifications.

Each section of this playground is compartmentalized and offers a set of challenges that can be completed to solidify your understanding of the angular practices.

To get to these exercises open the corresponding .html and scroll down until you see the Exercises Section of the html

The folder structure follows that of a typically Angular Application so your path to the exercises will look something like

app/components/home/home.html

On some occasions you will need to edit the controller associated with a view as well to complete an exercise. The controller can be found in the same pattern

app/components/home/HomeController.js
