Repeating html templates

One of the key features of Angular is the ability to easily repeat a snippet of html for each index in an Array or each property in an Object

In our controller we have an array of objects on the property data. Using a mix of Angular and HTML we can setup a template and then use the ng-repeat directive to do the rest of the work. However there are a few things to keep in mind.

Take a look at our player-card from the previous lesson


<div class="player-card">
	<button class="btn btn-xs btn-danger remove">Remove</button>
	<div>
		<img ng-src="player.photo" />
	</div>
	<div>
		<span>{{player.name}}</span>
	</div>
	<div>
		<span>{{player.position}}</span>
	</div>
	<div>
		<span>{{player.number}}</span>
	</div>
</div>

This snippet of HTML is currently setup to render a single player to the screen. Ideally we want all of the players to render using this same basic layout. To hand code each player would be incredibly tedious and labor intensive.

Now look at how we can utilize the ng-repeat directive


<div class="player-card" ng-repeat="player in controller.players">
	<button class="btn btn-xs btn-danger remove">Remove</button>
	<div>
		<img ng-src="player.photo" />
	</div>
	<div>
		<span>{{player.name}}</span>
	</div>
	<div>
		<span>{{player.position}}</span>
	</div>
	<div>
		<span>{{player.number}}</span>
	</div>
</div>

There are a few key features to point out when using the ng-repeat directive. The first is the order of words we have player in controller.players

First and formost we have the keyword player specified as the first word our value. This first word is arbitrary and could be called anything. For now just think of it as a variable for the value at i in a for loop


for(var i = 0; i < players.length; i++){
	var player = players[i];
}

Remember that controller.players is going to our controller and accessing the players property that has already been declared.

With this setup our html through the use of Angular knows to repeat this same section of html for each index in our players Array on the controller.
